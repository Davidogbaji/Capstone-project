# ✅ To-Do List App

A clean, modern task manager built with vanilla HTML, CSS, and JavaScript.  
Tasks are saved in your browser's `localStorage` – they survive page refresh.

🔗 **Live demo:** [https://Davidogbaji.github.io/Capstone-project](https://Davidogbaji.github.io/Capstone-project)

## ✨ Features

- ➕ Add new tasks  
- 🗑️ Delete tasks  
- ✅ Mark tasks as complete (click the task text)  
- 🔍 Filter by: All / Active / Completed  
- 💾 Persistent storage with `localStorage`  
- 📱 Fully responsive (works on mobile)  
- 🧹 Clean Git commit history
